# OwOifier
OwO's your text >~<
## Usage
Use `$` around text to mark them to not translate. For example, 
```
>>> "$Discord Kitten$: Thanks mr"
... "Discord Kitten: Thx mistuh"
```
## Future Updates
- Able to see, change, and delete settings within the app.
- Replace segments within text. For example, 
```
    >>> {"per": "purr"}
    >>> "Developer" 
    ... "Developurr" 
```
